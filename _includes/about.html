Sept 2018 – Nov 2018                                                         Amsterdam, NL
Next Empire (http://nextempire.net) is a digital product studio with many clients in the music business. For my audio segmentation project I’ve built from scratch an audio files classification model which seg- mented radiostreams into the individual songs and the speech or com- mercials in between. I used:
  • the HTML parser BeautifulSoup to retrieve data from remote websites;
  • librosa to decompose .mp3 files in spectrograms;
  • Python Keras to build the neural network classification model;
  • Changepoint Analysis to automate the classification process.
